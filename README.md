Proyecto de modulo 6 de e-commerce
Maven
Spring Boot
Spring Security
PostgreSql
RestAPI

Proyecto establece Base de datos con una tabla de productos y usuarios
Proyecto utiliza thymeleaf
Proyecto utiliza spring security para establecer un login con diferentes roles
Panel de Administrador que permite el agregar nuevos productos a la base de datos
Usuarios de prueba de Acceso: "admin" y "user"
Claves de prueba de acceso: "admin123" y "user123"

Instalacion y ejecucion de proyecto
Extraer contenido, Importar en eclipse, Ejecutar spring boot y acceder a la url de localhost
