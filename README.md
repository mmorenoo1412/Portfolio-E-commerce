# Portfolio-E-commerce
Demo Para certificacion de Bootcamp Talento Digital en base a un E-commerce
Herramientas Utilizadas
-Eclipse JDK
-Maven
-Spring Boot
-Spring Security
-JWT
-RestAPI
-Thymeleaf
-PostgreSQL para Base de datos
-HTML, CSS y JavaScript para FrontEnd

Funcionamiento de la Demo
-El proyecto cuenta con una tabla de productos
-Una simulacion de canasta de compras
-Un sistema de login de usuarios y usuarios administradores
-Sistema de registro de usuarios, a traves del cual un usuario nuevo puede ser creado y guardado en la base de datos
-Un panel de admin al cual solo puede acceder el administrador
-Un sistema de creacion de productos accesible por el administrador que puede añadir mas productos a la tabla de productos

Utilizacion del proyecto
-Descargar el comprimido
-Importar el proyecto en eclipse JDK
-Arrancar el proyecto a travez de spring boot

Navegacion del proyecto
-Al ingresar al port se verá el html de login
-Ingresar a un usuario ya sea administrador o usuario
-Claves de acceso | user: user123 | admin: admin123
-Alternativamente se puede registrar un usuario nuevo con rol de usuario
-Al ingresar a un usuario, se visualiza la pagina de productos con la tabla de productos
-Tambien se visualiza el navbar que puede usarse para acceder al carro de compras o volver a la pagina de productos
-Al acceder por usuario de Admin se visualiza tambien un boton de acceso al panel de admin el cual puede agregar productos
con un precio del producto a la tabla y el cual se diferencia por un cambio en la imagen de fondo
-Tambien se visualiza un boton de logout, el cual termina la sesion y redirecciona al panel de acceso de usuario
-En el carrito de compras se visualiza un boton de finalizar compra, el cual dado a que es una simulacion dará error 404
